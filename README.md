# Design Archive - 经典设计档案馆

一个致敬经典设计作品的数字档案馆，展示那些改变了产品设计、用户体验和视觉文化的标志性作品。

## 功能特性

- **响应式设计**：在移动端、平板和桌面设备上均有优秀表现
- **智能搜索**：实时模糊搜索，支持跨字段匹配
- **多维筛选**：按分类、年代、地区和排序方式筛选设计作品
- **收藏功能**：本地保存收藏的设计作品
- **暗黑模式**：支持浅色/深色主题切换
- **无障碍设计**：完整的ARIA标签、键盘导航和屏幕阅读器支持
- **图片懒加载**：优化页面加载性能
- **PWA支持**：可添加到主屏幕，支持离线访问
- **详细视图**：点击卡片查看设计作品的详细信息
- **无限滚动**：滚动加载更多设计作品

## 技术栈

- **HTML5**：语义化标记，无障碍支持
- **CSS3**：现代布局（Flexbox，Grid），CSS变量，自定义属性
- **JavaScript (ES6+)**：原生JavaScript，模块化设计模式
- **PWA**：Service Worker，Web App Manifest
- **响应式设计**：移动优先，自适应布局
